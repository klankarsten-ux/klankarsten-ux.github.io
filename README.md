# R10A Alumni – Jahrgang 1994

React/Vite-Webseite für die R10A der Heinrich-Heine-Schule Dreieich-Sprendlingen.

## Lokal starten

```bash
npm install
npm run dev
```

## GitHub Pages

1. Den gesamten Inhalt dieses Ordners in das Repository `klankarsten-ux.github.io` hochladen.
2. In GitHub unter **Settings → Pages → Build and deployment** bei **Source** „GitHub Actions“ wählen.
3. Nach dem nächsten Push wird die Seite automatisch veröffentlicht.

## Daten pflegen

Die öffentlichen Ausgangsdaten liegen in:

`public/data/classmates.json`

Der Adminmodus der Seite speichert Änderungen zunächst nur lokal im Browser. Mit **classmates.json exportieren** entsteht eine aktualisierte Datei. Diese ersetzt du anschließend im Repository unter `public/data/classmates.json`.

## Datenschutz

Das Repository und GitHub Pages sind öffentlich. Fotos, Wohnorte, Berufe und Social-Media-Hinweise sollten nur mit Einwilligung der Betroffenen veröffentlicht werden.
